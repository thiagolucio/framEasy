Fácil de entender, usar e implementar!

Usando a simplicidade do HTML5 você simplesmente não precisa de nada além do seu conhecimento pra usar.


Com as novidades estilosas do CSS3 você faz todo o resto necessário para uma aplicação com layout simples e bonito.
Um Framework que usa apenas HTML5 e CSS3, sem JavaScript nem Jquery. Se você procura uma solução simples para implementar em seu projeto, deixando ele bonito e leve sem grandes estruturas de pastas nem conflitos com seus códigos de JavaScript, essa é sem dúvida uma excelente opção.

Com apenas alguns passos que não leva mais do que 5 minutos você inclui em seu projeto este Framework amigável e 100% compatível com qualquer plataforma de desenvolvimento de projeto que estiver usando.
